# oshikryu.github.io

## Projects

- [Swimmingly](https://oshikryu.github.io/swimmingly) - Helps swimmers determine optimal times to swim at San Francisco's Aquatic Park by aggregating real-time data on tides, currents, weather, waves, water quality, and dam releases.

- [Case Strategy](https://oshikryu.github.io/case-strategy) - Helps O-1A visa applicants compile and organize evidence demonstrating extraordinary ability across eight criteria categories.

- [Spreadsheet Practice](https://oshikryu.github.io/spreadsheet-practice) - A spreadsheet application with editable cells, keyboard navigation, view/edit mode switching, and number formatting support.
